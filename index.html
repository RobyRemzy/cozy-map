Easy data mining on OpenStreetMap

This search is design to aggregate some Point Of Interst inside the map view (bbox). To help you to create a query you can check the documentation from Overpass.

Example:
[amenity=drinking_water]
[emergency=fire_hydrant]
[amenity=post_box]
[amenity=pub][microbrewery=yes]
[cuisine=japanese]
[amenity=fast_food][name="Big Fernand"]
[railway=station]
